# ACX-25 local provider fixtures

These Apache-2.0 project-authored cases describe attacks that a native restricted-provider profile must contain. ACXD-033 admits no native profile in draft 1, so conformance executes the deterministic preflight rejection before any case can launch.

The cases are data, never commands. Future positive profiles must implement separate inert workers and prove every case live on the exact claimed platform before changing support.
