# ACX-32 STEP/IGES fixtures

`ap214-xde.step` is generated by `generate_xde_fixture.py` with the exact reviewed OCP/OCCT image. It contains two project-authored box roots with names, colors, layers, physical materials, millimetre units and explicit placement. The fixture and generator are Apache-2.0 project assets and contain no third-party model content.

The ACX-17 AP203/AP214/AP242/IGES fixtures remain the source-profile corpus. ACX-32 reuses them by digest and adds this XDE-positive input plus deterministic replay/negative event fixtures; it does not broaden supported schemas.
