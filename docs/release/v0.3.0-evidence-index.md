The machine-readable authority is `conformance/v0.3/claims.json`, hash-bound by `conformance/v0.3/corpus.json`. The release checker requires every claim to map exactly once to ACX-24 through ACX-37 evidence and rejects targets, duplicate mappings, digest drift, replay-only positive claims and a positive claim for blocked ACX-34.
