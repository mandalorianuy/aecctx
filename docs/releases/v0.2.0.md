# AECCTX 0.2.0

AECCTX 0.2.0 publishes the governed expansion line while retaining read support for valid v0.1 packages. Install the reviewed artifact directly:

```bash
python -m pip install https://github.com/mandalorianuy/aecctx/releases/download/v0.2.0/aecctx-0.2.0-py3-none-any.whl
```

The release adds v0.2 evidence records, bounded IFC 2D/georeferencing, DXF source semantics/3D, optional English OCR, mesh coordinate qualification, experimental STEP/IGES and R2000 DWG provider profiles, offline Ed25519 signing, a deterministic quality gate, and the optional `aecctx-inspector` Codex plugin. Every claim is bounded by `conformance/v0.2/corpus.json` and the evidence index.

RVT remains unsupported with deterministic opaque fallback. Vision/reconstruction, hidden source geometry, unlisted formats/versions/platforms, online trust, proprietary binary redistribution, consumer mappings, engineering approval and WoodFraming integration are not release claims.

Verify downloaded assets with `SHA256SUMS`; the SPDX 2.3 document records core and optional dependency boundaries.
