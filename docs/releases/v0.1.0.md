# AECCTX 0.1.0

AECCTX 0.1.0 is the first public functional release of the application-agnostic, evidence-preserving AEC context package and Python reference toolchain.

Install the core with `pip install aecctx==0.1.0`. Optional extras are `dxf`, `ifc`, `pdf`, `image`, `geometry`, `mcp`, and `all`.

Verified capabilities include deterministic directory/ZIP packages; schema/integrity/semantic validation; opaque ingest; query/diff/context; IFC4 conformance evidence; AC1032 DXF evidence; vector/raster PDF and image evidence; OBJ mesh evidence; deterministic SVG/GLB derivatives; and isolated Python plugin/MCP surfaces.

Known partial or unsupported areas are recorded in every output package. Direct DWG/RVT, STEP/IGES, mandatory OCR/vision, write-back, signing/authenticity and consumer mappings are not release claims.
