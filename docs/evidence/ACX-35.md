- explicit Ed25519 leaf/optional-intermediate/root paths terminating in policy-selected root fingerprints;
- complete signed base CRLs supplied offline, with revoked, stale and unknown states distinct;
- closed AECCTX trusted-time tokens with explicit target, TSA identity/path/status and archival threshold;
- independent multi-signature thresholds and exact-target countersignatures that never inflate that threshold;
- separate integrity, cryptographic, identity, lifecycle, trust, authorization and archival-time results;
- SDK and `verify-advanced-trust` CLI parity without network, host trust store or host clock.

No aggregate authenticity, genuineness, approval or safety boolean exists.
